# Byml Library

Modern **B**inary **Yml** IO library written in managed C#

## Usage

### Reading a Byml File

```cs
byte[] data = File.ReadAllBytes("path/to/file.byml");
Byml byml = Byml.FromBinary(data);
```

### Writing a Byml File

```cs
/* ... */

using MemoryStream ms = new();
byml.Write(ms);
```
